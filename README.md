
# Wendy Onyiuke · AI-Powered Workflow Design Portfolio

[![Live Demo](https://img.shields.io/badge/demo-live-green)](https://iamsalvation.github.io/wendy-onyiuke-portfolio/)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue)](https://github.com/iamsalvation/wendy-onyiuke-portfolio)

A professional portfolio website for **Wendy Onyiuke** — an AI-Powered Workflow Designer and Project Coordinator. The site showcases her expertise in AI operations, project management, workflow automation, and systems thinking.

## 🎨 Features

- **Bold, Feminine Design** — Warm terracotta, amber, emerald, and beige color palette
- **Full Responsive** — Optimized for desktop, tablet, and mobile devices
- **Social Media Ready** — Open Graph and Twitter Card meta tags for rich link previews
- **Case Study Modals** — Interactive modals that showcase problem → process → result
- **Booking Integration** — Google Appointment, WhatsApp, and Email contact options
- **Systems Thinking Section** — Visual explanation of the Input → Processing → Output workflow
- **Smooth Animations** — Fade-in effects and hover transitions
- **SEO Optimized** — Meta descriptions, keywords, and structured JSON-LD data
